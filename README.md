# clyph-x-actions

**A selection of my custom [ClyphX Pro](https://isotonikstudios.com/product/clyphx-pro/) user actions for Ableton Live**

See them in action(!) at [twitch.tv/odisfm](https://twitch.tv/odisfm)

Feel free to use [my template](https://github.com/odisfm/clyph-x-actions/tree/main/_template) to code your own actions :)

## Installation

For each action you'd like to install:

1. Locate the `.py` file:
![image](https://i.imgur.com/Ut1Gswy.png)

2. Download the file:
![image](https://i.imgur.com/aXjciiE.png)

3. Place the file in your user actions directory.
On Mac this is `'/Users/[username]/Music/Ableton/User Library/Remote Scripts/_user_actions'`, on Window it is `'\Users\[username]\Documents\Ableton\User Library\Remote Scripts\_user_actions'`.
